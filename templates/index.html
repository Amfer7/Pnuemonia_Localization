About This Project

This web application leverages deep learning to assist in the early detection of pneumonia using chest X-ray images. The model provides weakly supervised localization â€” highlighting regions most indicative of pneumonia without explicit bounding box labels during training.

Model Information

The underlying neural network is a convolutional model fine-tuned on a curated chest X-ray dataset. It generates class activation maps (Grad-CAM) to visually explain predictions and improve interpretability for clinicians.
